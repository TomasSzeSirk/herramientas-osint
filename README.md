# OSINT Toolkit — Clase de Ciberseguridad
 
Repositorio de herramientas y recursos para la clase de OSINT dictada en la materia de Taller de Ciberseguridad, Ingeniería en Informática.
 
**Última actualización:** 12 de mayo de 2026
 
---
 
## ¿Qué hay en este repo?
 
```
/
├── README.md                  # Este archivo
├── herramientas-generales.md  # Herramientas de uso general para OSINT
└── herramientas-argentina.md  # Fuentes y recursos específicos para Argentina
```
 
---
 
## ¿Para qué sirve?
 
Este repo acompaña la clase y funciona como referencia rápida durante y después de la cursada. No es una guía exhaustiva — es un punto de partida con las herramientas más útiles, ordenadas por categoría y con notas de uso.
 
---
 
## Contexto legal
 
Todo el contenido de este repo asume uso en contextos legítimos:
- Investigaciones autorizadas
- Pentesting con scope definido
- CTFs y entornos de práctica
- Auditorías con consentimiento del cliente
El uso de estas herramientas fuera de ese contexto puede implicar infracciones a la **Ley 25.326**, la **Ley 26.388** y artículos del Código Penal argentino.
